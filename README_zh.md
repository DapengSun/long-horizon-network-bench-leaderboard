本项目是 Long-Horizon-Network-Bench evaluation leaderboard 的 Web 前端，用于展示
模型排名、benchmark 对比和单次评测详情。

快速导航：[快速开始](#快速开始) · [本地预览](#可选本地预览) · [开发](#开发)

本仓库接收由 NetEval Pro harness 导出的真实评测结果，并用于公开榜单展示：

```text
Long-Horizon-Network-Bench case bank
  -> NetEval Pro harness
  -> run_report.json
  -> site/src/data/raw-runs/export_to_leaderboard_*.jsonl
  -> pull request
  -> review and publish
```

普通提交者只需要提交 `site/src/data/raw-runs/` 下的导出 JSONL 文件。不要提交生成后的
前端数据。

## 功能

- **Ranking** - 可排序榜单，支持 benchmark 类别过滤、模型搜索和列配置。
- **Dashboard** - 雷达图对比模型在不同网络任务类别上的表现。
- **Evaluation details** - 查看指定模型和类别的 case 级结果，以及单 case 优化趋势图。
- **Localization** - 英文和中文 UI。
- **External link** - 跳转到 NetEval Pro，运行自己的评测。

## 项目结构

```text
leaderboard-mvp/
└── site/                 # React + Vite frontend
    └── src/
        ├── components/   # Ranking table, dashboard, detail views, site header
        ├── data/         # Raw runs, generated payloads, and metadata
        ├── features/     # Ranking and chart data helpers
        └── i18n/         # EN / ZH UI strings
```

## 快速开始

本地运行 NetEval Pro 后，将完成的 harness 运行导出到本 leaderboard checkout：

```bash
export LEADERBOARD_FRONTEND=/path/to/long-horizon-network-bench/leaderboard-mvp
export LEADERBOARD_DATA_DIR="$LEADERBOARD_FRONTEND/site/src/data"
export RUN_ROOT=/path/to/NetEval-Pro/.netops-runs

uv run neteval export leaderboard \
  --run-root "$RUN_ROOT" \
  --out-data-dir "$LEADERBOARD_DATA_DIR"
```

导出器会写入类似文件：

```text
site/src/data/raw-runs/export_to_leaderboard_<timestamp>.jsonl
```

在本仓库发起 PR。PR 应只包含 NetEval Pro 导出的 raw-runs JSONL 文件。

提交：

- `site/src/data/raw-runs/export_to_leaderboard_<timestamp>.jsonl`

不要提交：

- `.netops-runs/`
- 本地日志或 secrets
- `site/src/data/generated/`
- `site/dist/`

PR 描述建议：

```markdown
## Evaluation

- Model:
- Provider:
- Agent backend:
- Benchmarks / cases:
- Evaluation time:

## Source

- Generated by NetEval Pro: yes
- Raw-runs files:
  - `site/src/data/raw-runs/export_to_leaderboard_<timestamp>.jsonl`

## Notes

- Any known failed or skipped cases:
```

## 可选本地预览

提交 PR 不要求本地预览。如需先查看页面效果：

```bash
cd "$LEADERBOARD_FRONTEND/site"
npm install
npm run generate:data:real
npm run dev
```

这会生成 `site/src/data/generated/`；不要把该目录提交到 PR。

本地 production build：

```bash
npm run build:real
```
